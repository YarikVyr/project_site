# Магазин одежды ROFLS

## Скачивание

```bash
git clone https://github.com/YarikVyr/Project.git
```

## Библиотеки

```bash
pip install Flask
pip install sqlite3
```

## Инициализация библиотек

```
import sqlite3
from flask import Flask, render_template, request, redirect, url_for
```

## Запуск проекта

```bash
cd {путь маршрут до папки} phyton3 run app.py
```
